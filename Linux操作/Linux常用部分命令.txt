Linux的使用：
	1.Linux/Unix命令
	    命令名　　[选项]　　[参数]
	    注意：[]的内容代表可有可无
	2.Linux常用命令
		1.clear命令：清屏（Ctrl+L）
    	2.pwd  命令：用于显示当前的工作路径位置（当前工作文件夹）
	    3.ls　  命令：显示文件或文件夹的内容
            格式：ls　[选项]　[文件夹名或文件名]
            常用选项:
				-l    列表显示文件的详细信息
                -a    显示全部文件／文件夹
	    4.cd　  命令：改变当前工作路径（进入某个目录）
            格式：cd　路径
	    5.mkdir命令：创建一个或多个文件夹
	        格式：mkdir [选项] 文件名1　文件名2...
            常用选项：
				-p　如果是中间的文件夹不存在，则逐级创建所有文件夹
	    6.rmdir命令：删除一个或多个文件夹（文件夹必须为空）  
	        格式：rmdir [选项]　文件名
            常用选项：
				-p　如果是中间文件夹也不空，则逐级删除中间文件夹   
	    7.tree 命令：显示目录树
	    8.touch命令：
			1.创建新的文件
            2.如果文件已经存在，则更新文件的修改时间
            	格式：touch　文件路径
	    9.rm　  命令：删除文件或者文件夹
	        格式：rm [选项]　文件/文件夹
            常用选项：
				-r    递归删除文件夹内的文件和文件夹
	            -i    删除前给出提示（y=yes,n=no）
                -f    强制删除，不给出任何提示

Linux/Unix下的路径：
	路径：用来记录一个文件或文件夹的字符串
    	1.绝对路径：以'/'字符开头的路径为绝对路径(任何时候，一个文件的绝对路径都是唯一的）
        2.相对路径：
			1) 文件/文件夹名称
            2) .当前文件夹
            3) ..上一级文件夹
            4) ~用户主目录（home目录）
	1.操作带有特殊符号的路径的方法
	    用两个双引号（""）把路径括起来
    		e.g.:mkdir"a b"

help选项：查看此命令的帮助信息
    格式：命令名　--help
    	e.g.:ls --help

man命令帮助：查看命令所对应的帮助文档（手册）
    格式：man Linux/UNIX命令名
	    e.g.：man ls
    翻页：上下键
    退出：q键

快捷键：
	Ctrl+Alt:释放鼠标光标
    Ctrl+Alt+Enter:全屏/退出全屏（切换）
    Tab:补全命令和路径
    上下键:翻出以前的操作过的命令

sublime的快捷操作
	1.打开和隐藏侧边栏
    	view -> side　bar ->show/hide　side　bar
        Ctrl+k  b
	2.字体的放大和缩小
        Ctrl+'+'/'-'
	3.创建和取消多个光标
	    创建：Ctrl+鼠标左键单击
        取消：ESC键    
	4.多行注释/取消注释
	    Ctrl+/
	5.交替换行
	    Ctrl+shift+上下方向键
	6.多选相同的块
	    Ctrl+d
	7.cat命令:将文件内容作为标准输出打印到终端
	    cat　文件名1　文件名2
	8.more命令:分页显示文本文件内容
	    more　文件名
    	基本操作:
	        q     退出
        	空格  下翻一页
	9.通配符：
	    1.*   匹配０个，１个或多个任意字符
    	2.?   匹配１个任意字符
    	e.g.:
        	touch a ab ac abc aabb bc cd
	        1.ls a?   #匹配以a开头,文件名只有两个字符的文件
        	2.ls a*   #匹配以a开头的文件
	10.cp命令:复制文件或文件夹
	    cp　[选项]　源文件或文件夹　目标文件或文件夹
    	常用选项
            -a　复制文件夹中的所有内容
	11.mv命令:文件搬移或更名
	    mv 源文件或文件夹　目标文件或文件夹
	    e.g.:
    		mv passwd mypw  #改名字
			mv passwd ~/    #移动文件
	12.find命令:根据文件名等信息查找指定的文件
	    find　路径　-name　"文件名"
	13.grep命令:查找文件中相应的内容及文本信息
	    grep　”内容“　[选项]　文件名或路径
		常用选项
	        -n      显示行号
            -r/-R   递归搜索文件夹的文件

压缩和解压缩命令
	14.gzip命令:用zip压缩算法对文件进行压缩，生成压缩后的文件        
		gzip　文件名    
			.gz  为用gzip压缩过的文件    
	15.gunzip命令:用gzip压缩过的文件进行解压        
		gunzip　文件名
	16.tar命令:对文件或文件夹进行打包和解包的操作    
		tar [选项]　文件名　[文件名或路径]    
		常用选项：
	        -c  　　 		创建包
         	-x      	解包
	        -v   	 	显示操作的文件细节
	        -z   	 	用gzip/gunzip 对包进行压缩和解压
	        -f 文件名　 	正在操作的文件名
	        -c　路径　 	改变解压路径（只对解包有效）
        常用方法
			-czvf : 打包
	        -xzvf : 解压
								
		1、*.tar
			用 tar -xvf 解压					
		2、*.gz
			用 gzip -d或者gunzip 解压				
		3、*.tar.gz和*.tgz
			用 tar -xzf 解压
		4、*.bz2
			用 bzip2 -d或者用bunzip2 解压
		5、*.tar.bz2
			用tar -xjf 解压			
		6、*.Z 
			用 uncompress 解压
		7、*.tar.Z 
			用tar -xZf 解压		
		8、*.rar 
			用 unrar e解压
		9、*.zip 
			用 unzip 解压
	17.sudo命令 : 用超级用户权限来执行这些命令
	    sudo 命令 [选项]　[参数]
		常用选项
	        -i 切换到root用户
	18.exit命令 : 退出用户登录
	19.文件权限管理命令 : 查看文件权限
        ls -l        
		文件类型：            
			d  		文件夹            
			-　　 	普通文件            
			l 　 	链接文件
            c/b  	设备文件        
	20.文件权限            
		r　　读权限            
		w　　写权限            
		x　　执行权限
	21.权限分组    	
		用户权限(user) 
		组权限(group) 
		其它权限(other)
	22.root用户对一切拥有最高权限
		最高权限：rwx　rwx　rwx	#777
		最低权限：---　---　---	#000
		属主：owner
		属组：group
	23.chmod命令:修改文件权限    
		chmod 权限　文件名/文件夹名    
	    	权限
		        u　用户（属主）
        		g　同组用户
		        o　其它用户        
				a 所有用户        
				+　添加权限        
				- 删除权限
		e.g.:        
			chmod u+r   a.txt        
			chmod o-rwx a.txt
	24.输出重定向：
	    输出分两种        
			１．标准输出        
			２．标准错误输出
		1.标准输出重定向        
			>　将一个命令的标准输出重定向到指定文件        
			>> 将一个命令的标准输出重定向"追加"到指定文件    
		2.标准错误输出重定向        
			2>　将一个命令的标准错误输出重定向到指定文件        
			2>> 将一个命令的标准错误输出重定向"追加"到指定文件    
		3.重定向所有输出(include 1&2)        
			＆>        
			&>>
		4.丢进虚拟文件中，same as delete
			2> /dev/null
